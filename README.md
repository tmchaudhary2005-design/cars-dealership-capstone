# Cars Dealership - Full-Stack Capstone Project

## Project Overview
This project is a responsive web application for a national car retailer in the U.S. It allows users to view dealership branches across the country and submit reviews. 

## Technologies Used
* **Frontend:** React, HTML, CSS
* **Backend:** Django, Flask, Node.js
* **Databases:** MongoDB, SQLite
* **Deployment:** Docker, Kubernetes, IBM Cloud Code Engine
